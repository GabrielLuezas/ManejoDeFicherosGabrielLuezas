Cosas a tener en cuenta de la aplicación

-Fichero de configuración:

El fichero de configuración contiene 3 líneas :

formatodearchivo - Cada numero representa un tipo de fichero (0 - Por defecto, te muestra el menú de elección de archivo, 1 - TxT , 2 - Binario normal, 3 - Objetos binario, 4 - Acceso aleatorio binario, 5 - XML)

nombrecarpetaguardadoficheros - Nombre de la carpeta (Recomendable no poner espacios)

rutaguardadoficheros - Ruta donde se va a guardar la carpeta (A la hora de ponerla si la copias y la pegas poner doble \\, Ejemplo: C\:\\Users\\User\\Desktop)

-Propia Aplicación:

Cuando arranques la aplicación si no has tocado el fichero de configuración por defecto la carpeta se llamara Ficheros y se guardara en la carpeta del proyecto (A la altura de src)

El fichero de configuración también se encuentra a la altura de src

Para que se guarde el sistema de archivos tienes que usar la opción 7 del menú para salirte, si paras la ejecución del programa desde el propio IDE no funcionara
